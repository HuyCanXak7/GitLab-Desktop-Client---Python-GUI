# GitLab-Desktop-Client---Python-GUI
GitLab Desktop Client là ứng dụng GUI bằng Python, được thiết kế để giúp người dùng tương tác trực tiếp với GitLab (mặc định git.rikkei.edu.vn) mà không cần sử dụng trình duyệt. Ứng dụng thích hợp cho học sinh, sinh viên, và lập trình viên muốn thao tác nhanh trên project GitLab thông qua giao diện đồ họa.
